FLUJO DE ARQUITECTURA MVC
═══════════════════════════════════════════════════════

Frontend (VIEW)
    │
    │ HTTP Request (JSON)
    ▼
Routes (ROUTER)
    │
    │ Mapea a controlador
    ▼
Controller (CONTROLLER)
    │
    │ Delega lógica
    ▼
Service (BUSINESS LOGIC)
    │
    │ Opera con ORM
    ▼
Prisma (MODEL / ORM)
    │
    │ Ejecuta SQL
    ▼
Database (MySQL / PostgreSQL)

═══════════════════════════════════════════════════════
ARCHIVOS CLAVE
═══════════════════════════════════════════════════════

📄 backend/.env
   → Configuración de base de datos
   → Cambiar DATABASE_PROVIDER entre mysql/postgresql
   → DATABASE_URL con credenciales

📄 backend/prisma/schema.prisma
   → Modelo de datos (Product)
   → Define estructura de tablas
   → Compatible MySQL y PostgreSQL

📄 backend/src/services/productService.js
   → Toda la lógica de negocio de productos
   → Validaciones
   → Operaciones CRUD con Prisma

📄 frontend/src/App.jsx
   → Estado de la aplicación
   → Operaciones CRUD (crear, editar, eliminar)
   → Renderiza lista o formulario

📄 frontend/src/components/ProductForm.jsx
   → Formulario para crear/editar
   → Validación de campos
   → Comunicación con API

═══════════════════════════════════════════════════════
TECNOLOGÍAS UTILIZADAS
═══════════════════════════════════════════════════════

Backend:
  ✅ Node.js v18+
  ✅ Express 4.x (Framework web)
  ✅ Prisma 5.x (ORM)
  ✅ dotenv (Variables de entorno)
  ✅ cors (Cross-Origin Resource Sharing)

Frontend:
  ✅ React 18 (UI Library)
  ✅ Vite 5.x (Build tool)
  ✅ CSS vanilla (Sin frameworks)

Base de Datos:
  ✅ MySQL 8+ (Desarrollo)
  ✅ PostgreSQL 14+ (Producción)

═══════════════════════════════════════════════════════
COMANDOS PRINCIPALES
═══════════════════════════════════════════════════════

Backend:
  npm install              # Instalar dependencias
  npx prisma migrate dev   # Migrar base de datos
  npx prisma generate      # Generar cliente Prisma
  npm run dev              # Iniciar servidor (desarrollo)
  npx prisma studio        # Ver base de datos en navegador

Frontend:
  npm install              # Instalar dependencias
  npm run dev              # Iniciar app (desarrollo)
  npm run build            # Compilar para producción
